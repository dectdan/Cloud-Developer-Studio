Quick Start

After installation, open your terminal:

# Initialize your project
cd C:\Projects\MyApp
claudedev init .

# Start coding with Claude
# ClaudeDevStudio tracks everything automatically

# Next session - load context
claudedev load .

That's it! Claude now has persistent memory for your project.

System Requirements

Windows 10 (1809+) or Windows 11 • .NET 8.0 (auto-installed) • 50 MB disk space
